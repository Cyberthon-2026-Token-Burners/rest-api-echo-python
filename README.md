# rest-api-echo-python

A lightweight, high-performance Python 3.12 REST API echo server designed for cloud environments like GCP Cloud Run. It provides real-time HTTP mirroring of headers, payloads, and query parameters, and tracks process uptime via a stateless health endpoint.

## Features
- **Dynamic HTTP Echoing**: Mirrors request headers, payload bodies, and query parameters securely across GET, POST, PUT, PATCH, and DELETE.
- **Payload Size Protection**: Rejects requests exceeding a configurable byte-limit threshold (`MAX_PAYLOAD_SIZE` defaults to exactly 5,242,880 bytes / 5.0 MB) with HTTP 413, terminating connections before RAM exhaustion.
- **Custom Response Override**: Employs the `X-Echo-Status` header to allow API users to override and mock downstream response codes (ranges 100-599).
- **High-Performance Lifespan Health Check**: Tracks container uptime calculated since process initialization with microsecond accuracy.
- **Strict Input Validation**: Rejects invalid headers and configuration types early returning HTTP 400.

## Getting Started

### Prerequisites
- Python 3.12 or higher.

### Installation & Build
```sh
pip install --target=/workspace/.sdlc_deps -r requirements.txt 2>/dev/null || true
python -m compileall -q -x '(^|/)\.sdlc_deps(/|$)' .
```

## Usage
Start the Uvicorn server with environmental variables:
```sh
PORT=8080 MAX_PAYLOAD_SIZE=5242880 uvicorn app.main:app --host 0.0.0.0 --port 8080
```

## Running Tests
```sh
python -m pytest
```
