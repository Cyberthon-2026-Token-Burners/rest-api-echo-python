# rest-api-echo-python

A lightweight, high-performance Python 3.12 REST API echo server designed for cloud environments like GCP Cloud Run. It provides real-time HTTP mirroring of headers, payloads, and query parameters, and tracks process uptime via a stateless health endpoint.

## Features
- **Dynamic HTTP Echoing**: Mirrors request headers, payload bodies, and query parameters securely.
- **Payload Size Protection**: Rejects requests exceeding a configurable byte-limit threshold (`MAX_PAYLOAD_SIZE`) with HTTP 413.
- **High-Performance Lifespan Health Check**: Provides server uptime analytics calculated since process initialization.
- **Cloud-Ready Configurations**: Binds automatically to port environments defined under GCP Cloud Run specifications.

## Getting Started

### Prerequisites
- Python 3.12 or higher.

### Installation & Build
```sh
pip install --target=/workspace/.sdlc_deps -r requirements.txt 2>/dev/null || true
python -m compileall -q -x '(^|/)\.sdlc_deps(/|$)' .
```

## Usage
Start the Uvicorn server:
```sh
PORT=8080 MAX_PAYLOAD_SIZE=5242880 uvicorn app.main:app --host 0.0.0.0 --port 8080
```

## Running Tests
```sh
python -m pytest
```
